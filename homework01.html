1. Copy the hello_mpi.cpp program to pleiades and make sure you can compile and run it. There is nothing to turn in for this step, but it is recommended to make sure you can compile and run MPI programs successfully.
  2. Write an MPI program named hello_name.cpp which does the following:
    • Ask the user to enter his name. You may limit the input to 100 characters.
    • Send the user name from the root process to each other MPI process.
    • Each process (including the root process) should print Hello, name, from process N! where name is the name entered by the user and N is the rank of the process.
    Note: You send a string just as an array of characters. In this case, the buffer parameter would be a pointer to the start of the array, the type parameter would be MPI_CHAR, and the count character would be the number of characters you are sending. (Don't forget the null character.)

    For example, you could send a C-string like this: 

    char name[100] = "Suzy";

MPI_Send(name, 100, MPI_CHAR, destination, tag, MPI_COMM_WORLD);
  3. Write an MPI program maned avg_dist.cpp which determines the average distance of a group of experimental numbers from a target number. For example, if our experimental numbers are { 4, 11, 18 } and our target number is 10, then the average distance is (6 + 1 + 8) / 3 = 5.

    You must solve this problem as follows:

    • The user should first input the target number. (Do not print a prompt; let's assume the user knows what he is doing.)
    • After that, the user should enter as many experimental numbers as there are processes. (You are programmers; you should be able to whip a program that spits out random numbers.)
    • The root process should send each other process the target number and one experimental number. (It will need to keep one for itself.)
    • Each other process should then calculate the distance from its experimental number to the target number and send it back to the root process. (The distance should always be positive.)
    • The root process should average the results and then print out the answer. (I couldn't think of anything to put in parentheses for this one.)
